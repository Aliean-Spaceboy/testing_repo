# 🇩🇪 Deutsches Tagebuch – German Practice Diary

A beautiful, dark-mode web app to practice German daily and track your journey from A1 to B2.

## ✨ Features

| Feature | Details |
|---|---|
| 📓 **Daily Diary** | 3 guided prompts every day (What did I do? What did I learn? What will I do tomorrow?) |
| 📚 **Vocabulary Tracker** | Add 10 words/day with category tags (Tech, Work, Daily, Verbs, etc.) |
| 🃏 **Flashcard Tester** | Random flashcard quiz from your own word list |
| 📊 **Progress Dashboard** | Streak counter, level badge (A1→B2), activity heatmap |
| 🎤 **Speaking Prompts** | Weekly topics with hint phrases to practice speaking aloud |
| ✏️ **Grammar Tips** | 20 essential grammar rules with examples |
| 💾 **Local Storage** | All data saved in your browser — no account needed |

## 📅 Daily Routine (Use This App For)

```
Morning   → Think in German: "Heute lerne ich Java."
Evening   → Open the app, write all 3 diary prompts (10 min)
Any time  → Add 10 vocabulary words to your list
Weekly    → Use a speaking prompt, record yourself
```

## 🎯 Level Milestones

| Level | Score Target | What You Can Do |
|---|---|---|
| A1 | 50 pts | Introduce yourself, basic phrases |
| A2 | 150 pts | Daily conversations, simple emails |
| B1 | 350 pts | Work discussions, meetings |
| B2 | 600 pts | Interviews, technical discussions |

_Score = (diary entries × 5) + vocabulary words_

## 💡 Tech Stack

- Pure HTML + CSS + JavaScript (no frameworks, no dependencies)
- All data stored in browser's `localStorage`
- Zero backend required — works completely offline
